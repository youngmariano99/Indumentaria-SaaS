# Sistema de Documentación - Indumentaria-SaaS

Bienvenido al Manual Definitivo del Sistema. Este repositorio de conocimiento es la fuente oficial de verdad para entender, mantener y expandir la plataforma.

## 🧭 Mapa de Navegación

1.  **[Arquitectura Core](01-Arquitectura-Core.md)**: El "Cerebro" del sistema. Clean Architecture, Seguridad RLS y Motor Multi-Rubro.
2.  **[Filosofía UX/UI](02-Filosofia-UX-UI.md)**: El "Corazón" del sistema. UX Educativa, PWA, Mobile-First y UI Mutante.
3.  **[Mapa de Módulos](03-Mapa-Modulos.md)**: El "Cuerpo" del sistema. Detalle de capacidades actuales y sus conexiones.
4.  **[Guía de Nuevos Módulos](04-Guia-Nuevos-Modulos.md)**: El "Manual de Construcción". Pasos técnicos para añadir nuevas funciones sin desentonar.

---

## 🎯 Objetivo
Garantizar que Indumentaria-SaaS crezca de forma coherente, profesional y escalable, eliminando la deuda técnica y asegurando una experiencia de usuario excepcional en cualquier rubro industrial.
